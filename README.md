## Overview.

This repository contains a react movie application which involves "The movie database" API. There were 4 initial labs to start the app and it was then extended for an assignment.

### Features.
+ Feature 1
  Popular movies page. This page displays current popular movies.
+ Feature 2
  Now playing movies page. This displays movies that are currently playing.
+ Feature 3
  Top rated movies page. This displays movies with the highest rating in the database.
  Adding a movies to favourites is still a feature of these thee new features.
+ Feature 4
  Recommenedations button added to the card component. This links to the recommendations page. (To be completed)
+ Feature 5
  Recommendations page: Displays movie recommendations based on a selected movie. (To be completed)
+ Feature 6
  Labbeled with a new MUI icon of a play button, I added video content to the card component. (To be completed)

## Setup requirements.
Must make sure the API key is setup corrently to enable access to the movie database.

## API endpoints.
+ Popular movies - movies/popular
  This page displays current popular movies.
+ Now playing movies - movies/nowPlaying
  This displays movies that are currently playing.
+ Top rated movies - movies/topRated
  This displays movies with the highest rating in the database.
+ Recommeneded movies - movies/:id/recommendations
  Displays movie recommendations based on a selected movie.

## Routing.
+ movies/popular
+ movies/nowPlaying
+ movies/topRated
+ movies/:id/recommendations
